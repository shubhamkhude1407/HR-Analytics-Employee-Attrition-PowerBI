# 📊 HR Analytics Employee Attrition & Performance Dashboard

An interactive Power BI dashboard designed to analyze employee attrition, workforce performance, and HR metrics. The dashboard enables HR professionals and business leaders to identify workforce trends, monitor key performance indicators, and support data-driven decision-making.

---

# 📌 Project Overview

Employee attrition is one of the biggest challenges for organizations. This project provides an interactive HR Analytics dashboard that helps identify patterns in employee turnover and workforce distribution through clear visualizations and KPIs.

---

# 🎯 Objectives

- Analyze employee attrition trends.
- Monitor workforce demographics.
- Identify high-risk employee segments.
- Provide actionable insights for HR decision-making.
- Create an interactive dashboard for business users.

---

# 🛠 Tools & Technologies

- Microsoft Power BI
- Power Query
- DAX
- CSV Dataset

---

# 📂 Dataset

The dataset contains employee-related information such as:

- Employee Details
- Department
- Job Role
- Gender
- Age
- Education
- Salary
- Attrition Status
- Other HR attributes

> Dataset included in this repository for learning and portfolio purposes.
